How to make a webpage

  1. Create a file ending with html.
  2. Log into your sftp account.
  3. Start typing and do some coding in the html file.
  4. You should validate your coding periodically, so that if you mess up somewhere, it will be easier to fix.
  5. Once you're done, you can view it on a development server by using the command python -m SimpleHTTPServer.
  6. If you're happy with what you have, then you can upload it to sftp using the put command.
  7. Upload it to the sftp server by using the command sftp pat1748@cmis-hsteacher.
  8. To save to your repository, you can use the command git commit -m "".
  9. Finally, use the command git push to save to your repo.
  10. Make sure to check in your repository that you have done all the procedures correctly, so as not to lose your work.
